═══════════════════════════════════════════════════════════════════════════════
                              CÓMO FUNCIONA
═══════════════════════════════════════════════════════════════════════════════

FLUJO PARA JUGADOR NORMAL (Convocado):
1. Abres detalle del jugador
2. Ves todos los partidos registrados
3. Los partidos ya pagados muestran: "vs Rival ✓ Pagado" [fondo verde]
4. Seleccionas los partidos a pagar
5. Sistema calcula automáticamente (según estado: T, S, S c/min)
6. OPCIONALMENTE: Editas el monto en "💰 Monto a pagar (editable)"
7. Si seleccionaste partidos ya pagados → aparece advertencia
8. Confirmas y se registra el pago

FLUJO PARA JUGADOR NO CONVOCADO (Arreglo Especial):
1. Abres detalle del jugador
2. Los partidos muestran: "vs Rival No convocado" [badge rojo]
3. Checkbox está HABILITADO (puedes seleccionar)
4. Seleccionas el/los partido(s)
5. Sistema NO calcula monto automático (= $0)
6. OBLIGATORIO: Ingresas el monto en "💰 Monto a pagar (editable)"
7. Confirmas y se registra el pago

                            CASOS DE USO
═══════════════════════════════════════════════════════════════════════════════

📌 CASO 1: Verificar si un jugador fue pagado
   → Abre el detalle y busca "✓ Pagado" en verde

📌 CASO 2: Pagar a un jugador que ya fue pagado (corrección)
   → Selecciona el partido, aparece advertencia, confirmas y continúas

📌 CASO 3: Pagar a jugador no convocado (préstamo, arreglo)
   → Selecciona el partido N/C, ingresa monto manual, guarda

📌 CASO 4: Ajustar un monto calculado
   → Deja que calcule automáticamente, luego edita en "💰 Monto a pagar"
   → Ej: cálculo = $500, cambias a $550 para bonus especial

📌 CASO 5: Pagar múltiples partidos con monto flat
   → Selecciona varios partidos, calcul automático suma, edita si necesitas

                          DATOS TÉCNICOS
═══════════════════════════════════════════════════════════════════════════════

📊 Fuente de datos de pagos previos:
   → Hoja Google Sheets: "Historial_Pagos"
   → Variable JS: allPagos (se carga al iniciar)
   → Storage local: localStorage.getItem('pagos_historial')

📊 Cómo se guarda un pago:
   → Se envía a Google Sheets (tabla Historial_Pagos)
   → Se actualiza allPagos en memoria
   → Se usa para futuras verificaciones
